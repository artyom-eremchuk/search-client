# Запуск Backend

1. Зайти в директорию backend

---

2. Установить зависимости

```bash
$ npm i
```

3. Запустить сервер

```bash
$ npm run dev
```

---

- Сервер работает на порту `5000`

# Запуск Frontend

1. Зайти в директорию frontend

---

2. Установить зависимости

```bash
$ npm i
```

3. Запустить сервер

```bash
$ npm run dev
```

---

- Сервер работает на порту `5173`
- Доступно по адресу `http://localhost:5173`
